# Integrations

InsightPulse AI connects multiple services through well-defined integration points.

## Integration Map

| Source | Target | Method | Purpose |
|--------|--------|--------|---------|
| Odoo | Foundry | REST (Copilot Gateway) | AI chat, agent invocation |
| Odoo | Databricks | JDBC extract | Analytics data pipeline |
| Odoo | Document Intelligence | REST | Receipt/invoice OCR |
| n8n | Odoo | XML-RPC | Workflow automation |
| n8n | Slack | Webhook | Notifications, ChatOps |
| n8n | Supabase | REST | Task bus, external events |
| Plane | Odoo | Bidirectional sync | Issues ↔ Tasks |
| Superset | Databricks SQL | JDBC | Dashboard queries |

## MCP Servers

Model Context Protocol servers provide tool access to AI agents:

| Server | Status | Purpose |
|--------|--------|---------|
| `plane` | Live | Project management via Plane.so API |
| `odoo-erp` | Planned | Odoo CRUD via FastAPI endpoints |
| `databricks` | Planned | SQL queries, job triggers |

## n8n Automation

Self-hosted n8n at `n8n.insightpulseai.com` handles:
- Slack ChatOps commands
- Scheduled data syncs
- Alert routing and escalation
- GitHub/Azure DevOps webhook processing
