# Checklist — databricks-production-readiness-judge

- [ ] Feature release maturity classified (GA / Public Preview / Beta / Private Preview / Experimental)
- [ ] Maturity source cited (release notes URL, docs page, changelog)
- [ ] Deployment path assessed (codified / partial / manual)
- [ ] Observability/evaluation evidence assessed (complete / partial / absent)
- [ ] Rollback/recovery expectations assessed (defined / partial / undefined)
- [ ] Lane-specific skill assessment referenced (pipeline/app/agent/model)
- [ ] Production-grade classification determined (production-ready / preview-acceptable / not-production-grade)
- [ ] Beta/Private Preview/Experimental correctly rejected as production baseline
- [ ] Adoption recommendation generated (adopt / adopt-with-conditions / defer / reject)
- [ ] Conditions listed explicitly (if adopt-with-conditions)
- [ ] Risk assessment completed
- [ ] Evidence collected and saved to docs/evidence/{stamp}/databricks/readiness-judgment/
