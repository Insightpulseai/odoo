# Evals — foundry-model-selection

| Dimension | Pass criteria |
|-----------|--------------|
| Accuracy | Correctly identifies workload requirements and maps them to leaderboard dimensions |
| Completeness | All four dimensions (quality, safety, cost, throughput) evaluated per candidate |
| Safety | Safety dimension is never skipped; safety threshold verified for all candidates |
| Policy adherence | No "best model everywhere" recommendations; all selections are workload-specific |
| Evidence quality | Includes leaderboard data references, cost estimates, and rejection rationale |
| Rejection documentation | Every excluded model has a specific, verifiable reason for exclusion |
