# Checklist — product-portfolio-judge

- [ ] Product-manager output received and validated (spec bundle inventory)
- [ ] Portfolio-manager output received and validated (goal hierarchy + capacity plan)
- [ ] Cross-reference matrix built (every spec mapped to goal, every goal mapped to spec)
- [ ] Orphan specs identified (specs with no parent goal)
- [ ] Unfunded goals identified (goals with no implementing spec)
- [ ] Priority consistency validated (spec priority aligns with goal priority)
- [ ] Timeline feasibility assessed (spec dates vs milestone dates)
- [ ] Stale items flagged (no activity > 30 days)
- [ ] Every item categorized as MATCHED, ORPHANED, or CONFLICTING
- [ ] Recommended actions produced for each non-MATCHED item
- [ ] No items auto-closed or auto-resolved (flag only)
- [ ] Evidence directory created at `docs/evidence/{stamp}/alignment/{review-period}/`
