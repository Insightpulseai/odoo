# Odoo.sh First Module Guide

## What it is

A guide to creating and deploying a custom module on Odoo.sh.

## Key concepts

- **Scaffolding:** `odoo-bin scaffold`.
- **Branching:** Create feature branch.
- **Testing:** Test on dev branch -> Staging -> Production.

## Patterns

1. Create branch.
2. Scaffold module.
3. Push to Odoo.sh.
4. Install module in build.
