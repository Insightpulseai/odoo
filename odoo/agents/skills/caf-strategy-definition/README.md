# caf-strategy-definition

Defines cloud adoption strategy with business justification, motivations, outcomes, and financial model using CAF Strategy phase methodology.

## When to use
- New cloud adoption initiative is proposed
- Business case review for cloud migration
- Annual cloud strategy refresh
- New workload class identified for cloud consideration

## Key rule
Every cloud adoption must start with a documented strategy that includes motivations, business outcomes, financial model, and digital estate rationalization. Strategy outputs inform architecture but do not override existing platform commitments.

Strategy
