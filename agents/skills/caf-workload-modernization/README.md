# caf-workload-modernization

Modernizes existing workloads to improve business outcomes using CAF Modernize phase methodology, including application modernization patterns, database modernization, containerization assessment, and PaaS migration.

## When to use
- Workload on IaaS identified for PaaS modernization
- Performance or scalability issues on existing workload
- Database modernization opportunity identified
- Containerization assessment requested
- Cost optimization through managed services

## Key rule
Modernization must be incremental (strangler fig or parallel run), never big-bang rewrites. Performance baselines must be captured before and after modernization to prove improvement.

## CAF Phase
Modernize
