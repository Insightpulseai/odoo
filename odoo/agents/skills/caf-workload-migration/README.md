# caf-workload-migration

Assesses and executes workload migration using CAF Migrate phase methodology, including workload assessment, migration wave planning, rehost/replatform patterns, and cutover/rollback planning.

## When to use
- Workload identified for migration to Azure
- Migration wave planning is being initiated
- Cutover window is scheduled
- Post-migration validation required

## Key rule
Every migration must have a documented rollback procedure tested before cutover. Data integrity validation is mandatory before decommissioning the source environment.

## CAF Phase
Migrate
