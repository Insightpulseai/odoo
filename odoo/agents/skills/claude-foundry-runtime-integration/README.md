# Skill: Claude Foundry Runtime Integration

## Purpose

Integrates Claude as a model provider within Azure AI Foundry. Covers Foundry model catalog deployment, Claude-specific configuration, and coexistence with other Foundry-hosted models.

## Owner Persona

`claude-runtime-integrator`

## Skill Type

`capability_uplift` — provides Foundry-specific Claude integration knowledge.

## Key Principle

Claude on Foundry is one integration path — not the only one. Direct API remains valid. The choice depends on governance requirements and existing Foundry investment.
