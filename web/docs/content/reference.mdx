## Module Inventory

Custom IPAI modules follow the naming convention `ipai_<domain>_<feature>`:

| Module | Domain | Purpose |
|--------|--------|---------|
| `ipai_finance_ppm` | Finance | Project profitability management |
| `ipai_odoo_copilot` | AI | Copilot chat bridge to Foundry |
| `ipai_tax_compliance` | Finance | BIR tax filing automation |
| `ipai_slack_connector` | Integration | Slack ChatOps bridge |
| `ipai_auth_oidc` | Security | OIDC authentication (Entra target) |
