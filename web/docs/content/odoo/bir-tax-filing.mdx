# BIR Tax Filing (Philippines)

InsightPulse AI supports Philippine tax compliance through the TaxPulse-PH sub-agent, which helps finance teams prepare, validate, and file BIR (Bureau of Internal Revenue) tax returns from Odoo on Cloud. It covers VAT, withholding tax, income tax, and documentary stamp tax workflows.

## Tax Computation Workflow

### VAT (2550M/2550Q)
1. **Output VAT**: Sum of VAT on all customer invoices posted in the period
2. **Input VAT**: Sum of VAT on all vendor bills posted in the period
3. **VAT Payable** = Output VAT - Input VAT (if positive)
4. **Excess Input VAT** = carried forward to next period (if negative)
5. Validate: cross-check with Sales Journal and Purchase Journal totals

### Expanded Withholding Tax (1601-E)
1. Extract all vendor bills with EWT tax lines
2. Group by ATC (Alphanumeric Tax Code) — e.g., WE010 (5%), WE050 (2%)
3. Summarize: tax base + tax withheld per ATC
4. Generate BIR 2307 certificates per vendor
5. File 1601-E with eFPS or eBIR

### Compensation Withholding (1601-C)
1. Extract payroll journal entries for the period
2. Compute: gross compensation, non-taxable, taxable, tax withheld
3. Apply BIR tax table (graduated rates)
4. File 1601-C

## FAQ

**Q: Does InsightPulseAI file directly with BIR?**
A: Not yet. TaxPulse-PH prepares the data, validates it, and generates the filing-ready summary. Actual filing is done through eFPS or eBIR by your tax team. Direct eFPS integration is on the roadmap.

**Q: How are tax credits and excess input VAT handled?**
A: Excess input VAT is tracked in a carrying account and automatically applied as a credit in the next period's 2550M computation.

**Q: Can it handle multiple branches or RDOs?**
A: Yes. Multi-company Odoo setup maps each branch to its BIR-registered RDO (Revenue District Office). Each company files independently.

**Q: Is AvaTax used for Philippine taxes?**
A: No. AvaTax does not cover the Philippine market. TaxPulse-PH is the dedicated Philippine tax engine. AvaTax is benchmarked as a future complement for international tax scenarios.
