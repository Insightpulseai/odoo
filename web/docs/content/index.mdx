## Architecture at a Glance

| Plane | Service | Purpose |
|-------|---------|---------|
| **Operational** | Odoo CE 19 | ERP system of record — finance, HR, projects, sales |
| **AI/Agent** | Azure AI Foundry | Agent runtime, LLM orchestration, MCP tools |
| **Analytics** | Databricks + Unity Catalog | Governed data engineering, lakehouse, ML |
| **Identity** | Microsoft Entra ID | Authentication, authorization, agent identity |
| **Runtime** | Azure Container Apps | Containerized workloads behind Front Door |
| **CI/CD** | Azure DevOps | Build, test, deploy pipelines |

## Stack Summary

- **ERP**: Odoo CE 19.0 + OCA community modules + custom `ipai_*` modules
- **Hosting**: Azure Container Apps (Southeast Asia region)
- **Database**: Azure PostgreSQL Flexible Server
- **AI**: Azure AI Foundry (GPT-4o, Document Intelligence, AI Search)
- **Data**: Databricks on Azure + Unity Catalog + DLT pipelines
- **Automation**: n8n (self-hosted) + Slack ChatOps
- **DNS/Edge**: Cloudflare DNS + Azure Front Door (TLS, WAF)
- **Domain**: `insightpulseai.com`
