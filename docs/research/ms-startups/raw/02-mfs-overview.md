# Two-track program (as of July 2, 2025)

Microsoft restructured in July 2025. There are now two distinct paths:

## Path A: Azure startup credit offer (open, no application)
- $1,000 immediate on MSA sign-in
- +$4,000 after business verification (total $5,000)
- Credits valid 90 days ($1k) / 180 days (verified unlock)
- Auto-converts to pay-as-you-go after credits exhaust or expire
- No Series/funding restriction for this tier
- Eligibility: new Azure customer, MSA, eligible country, registered legal business entity (for unlock), not gov/edu/consultancy/dev-shop/agency/crypto

## Path B: Microsoft for Startups investor offer (referral-gated)
- Starts at $100,000 Azure credits, "opportunity to earn more"
- Full benefits stack (see benefits file)
- Eligibility:
  - Affiliated with an investor in Microsoft for Startups Investor Network (list is NOT public)
  - Software-based product, owned not licensed
  - <$350k lifetime free Azure credits received
  - Headquartered in Azure-available country
  - Has NOT raised Series C
  - Privately held, for-profit
  - Not edu/gov/blog/dev-shop/consultancy/agency/crypto-mining

## Portal access (post-approval)
- portal.startups.microsoft.com/login
- Functions: update profile, add team members, activate benefits, request Azure technical guidance, submit support tickets

## Development stage classification (during application)
1. Concept design
2. Prototyping
3. Building MVP
4. MVP in market
5. Established market

## What's retired (July 2, 2025)
- Expert Network program
- Champion Network (brand and program)
- "Founders Hub" as a separate name — now unified as "Microsoft for Startups"
- (Pegasus program was explicitly NOT retired)
