# infra/azure/marketplace

Azure Marketplace deployment artifacts for **Pulser for Odoo — Finance Operations Control Tower**.

## Contents

| File | Purpose |
|---|---|
| `createUiDefinition.json` | Azure Marketplace "Deploy to Azure" UI form — collects deployment parameters via a guided wizard |

## UI Form Overview

The `createUiDefinition.json` defines a five-step deployment wizard:

| Step | Fields |
|---|---|
| **Basics** (auto-generated by Azure) | Subscription, resource group, region |
| **Odoo Configuration** | Admin email, database name, PostgreSQL admin password, PostgreSQL SKU |
| **AI & Analytics** | Enable Pulser AI, Foundry resource name, Enable Databricks |
| **Networking** | Enable private endpoints, custom domain |
| **Tags** | Environment (dev/staging/prod), organization name |

## Linked Deployment Template

This UI definition is paired with `infra/azure/platform-stamp.bicep` (subscription-scope deployment). The `outputs` block maps wizard fields directly to Bicep parameters.

## Testing Locally

Use the Azure portal sandbox to validate the UI definition before publishing:

```
https://portal.azure.com/#blade/Microsoft_Azure_CreateUIDef/SandboxBlade
```

Paste the contents of `createUiDefinition.json` into the editor to preview the wizard.
