# Shared UI Component Library

Reusable UI components shared across web properties. Components follow the InsightPulse AI design system and are framework-agnostic where possible.

## Convention

- Components are React + TypeScript
- Styling uses design tokens from `packages/brand/`
- Each component has a co-located test file
- Exported via package index for tree-shaking
