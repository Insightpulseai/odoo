# Services: Project, Timesheets, Field Service

## What it is

Core applications for service-based businesses.

## Key concepts

- **Project:** Task management (`project.project`, `project.task`).
- **Timesheets:** Time tracking (`account.analytic.line`).
- **Field Service:** On-site task management (extension of Project).

## Patterns

- **Analytic Accounting:** Projects link to Analytic Accounts for financial tracking.
- **Portal:** Customers can view tasks/timesheets in the portal.
