# Evals — caf-governance-baseline

| Dimension | Pass criteria |
|-----------|--------------|
| Accuracy | Correctly assesses governance maturity level with supporting evidence |
| Completeness | All 5 governance disciplines evaluated: cost, security, identity, resource consistency, deployment |
| Policy coverage | Recommended CAF baseline policies identified and gap analysis provided |
| Cost awareness | Cost analysis uses actual spend data and respects self-hosted philosophy |
| Evidence quality | Evidence includes policy assignment lists, compliance percentages, cost reports |
| Safety | Policies deployed as audit-first; no production-blocking deny policies without migration path |
