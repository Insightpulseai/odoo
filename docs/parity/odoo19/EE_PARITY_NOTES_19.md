# Odoo 19 vs IPAI Stack (EE-Parity Notes)

## Baseline

- Odoo core: **18.0 CE**
- Extensions: **OCA 18.0** + `ipai_*` bridge modules
- Target: **Enterprise-grade feature coverage for TBWA/W9 use cases**

## 1. What Odoo 19 Adds (High Level)

- General UX: activities, kanban, Gantt, list, mobile, portal, import templates, etc.
- Technical: cached data & translations, performance improvements.
- Integrations: Twilio SMS; richer email/connectors; AI-driven HTML fields, etc.

> Source: Official Odoo 19 release notes & editions comparison (mirrored in this folder).

## 2. Editions Matrix (Community vs Enterprise)

- Enterprise adds:
  - Unlimited functional support, version upgrades, hosting.
  - Studio, mobile apps, extra finance features (OCR, ESG, consolidations), etc.
- Community = core framework; Enterprise = Community + licensed extras.

## 3. IPAI Positioning

- "Built on **Odoo 18 Community** with **Enterprise-grade extensions**."
- "Feature parity with **Odoo Enterprise 18** for TBWA/W9 finance + ops stack."
- Not an **official Odoo Enterprise 19** installation or image.
