# Tool Routing Gateways

Tool routing gateways for agent-to-service calls.

## Responsibilities

- Route tool invocations from agents to backend services (Odoo, Databricks, Azure AI)
- Enforce authentication and authorization per tool
- Rate limit and circuit-break outbound calls
- Log tool call telemetry for observability
