## Architecture Overview

5-module architecture with clean separation of concerns:

1. **ipai_workspace_core** - Shared foundation (base, mail)
2. **ipai_ppm** - Portfolio/Program/Risk/Resource allocation (OCA integration)
3. **ipai_advisor** - Recommendations + 5-category scoring
4. **ipai_workbooks** - Superset analytics registry
5. **ipai_connectors** - n8n/webhooks/Mattermost integration
