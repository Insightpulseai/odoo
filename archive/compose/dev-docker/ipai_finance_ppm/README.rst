This module provides comprehensive Finance Project Portfolio Management (PPM)
capabilities for Odoo 18 Community Edition, including:

* **BIR Tax Calendar** - Philippine tax deadline tracking with automated reminders
* **Month-End Closing WBS** - Hierarchical Work Breakdown Structure for closing tasks
* **Preparer/Reviewer/Approver Workflow** - Multi-level approval process
* **Canvas Dashboards** - Configurable widgets with ECharts visualization

Features
========

Finance PPM Task Management
---------------------------

* Hierarchical task structure with parent-child relationships
* LogFrame levels: Phase and Task
* Role-based assignments: Preparer, Reviewer, Approver
* State workflow: Draft → In Progress → Review → Approval → Done
* Deadline tracking with offset calculations
* Full audit trail via mail.thread integration

Finance Team Directory
----------------------

* Team member registry with role codes (e.g., CKVC, RIM, BOM)
* Email integration for notifications
* Quick lookup via code-based naming

Canvas Dashboard System
-----------------------

* Customizable dashboard canvases with 4/6/8 column layouts
* Widget types: Chart, Table, Number Tile
* Chart types: Bar, Pie, Line, Ring (donut), Number gauge
* Dynamic data fetching with group_by and domain filters
* ECharts JSON generation for frontend rendering
* Governance limits: Max 10 widgets, max 7 tables per canvas

BIR Tax Calendar (Philippines)
------------------------------

Seed data includes 2026 BIR tax calendar with:

* 1601-C Monthly Withholding Tax
* 1601-EQ Quarterly Expanded Withholding
* 1702-RT/EX Annual Income Tax Return
* 2550Q Quarterly VAT
* Parent-child task hierarchy with sub-tasks for Prep/Review/Approve

Configuration
=============

No special configuration is required. After installation:

1. Navigate to **Finance PPM** menu
2. Create team members in **Finance Directory**
3. Create tasks and phases in **Tasks & Calendar**
4. Build dashboards in **Dashboards**

Usage
=====

Creating a Finance Task
-----------------------

1. Go to Finance PPM > Tasks & Calendar
2. Click "Create"
3. Enter task name and select parent phase (optional)
4. Assign Preparer, Reviewer, and Approver roles
5. Set deadline date
6. Save and use workflow buttons to progress through states

Building a Dashboard
--------------------

1. Go to Finance PPM > Dashboards
2. Click "Create" and name your canvas
3. Select layout (4, 6, or 8 columns)
4. Click "Add Chart Widget" to create widgets
5. Configure each widget:
   - Select target object (Tasks or Team)
   - Choose widget type (Chart, Table, Tile)
   - Set group_by field and operation (Count, Sum, Avg)
   - Add domain filter if needed
