# Modules

InsightPulse AI follows a strict module philosophy:

```
Config → OCA → Delta
```

1. **Config** — use built-in Odoo configuration first
2. **OCA** — adopt vetted community modules second
3. **Delta** — build custom `ipai_*` modules only for truly custom needs

The platform targets 60 modules across 8 functional layers, achieving >=80% Enterprise parity through the CE + OCA + ipai_* combination.
