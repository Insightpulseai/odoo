# sap-security-operations

Extracts security benchmark patterns from SAP on Microsoft Cloud guidance.

## When to use
- Security architecture review for ERP/platform workloads
- Entra identity integration design
- SIEM/SOAR pattern design
- Monitoring architecture for Azure workloads

## Covers
- Entra ID integration patterns
- Security Copilot benchmark extraction
- Sentinel monitoring patterns
- SOAR response patterns
- Azure monitoring for workloads
