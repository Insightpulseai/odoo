IPAI Control Room

A comprehensive operations control center for managing pipelines, data quality, and operational insights.

Features

Pipelines

ETL/automation workflows with stages, steps, and execution runs. Supports manual, scheduled, webhook, and event triggers.

Data Quality

Define rules, run checks, and track issues. Supports various rule types including not null, unique, range, schema, and custom SQL.

Advisor

AI-powered recommendations and playbooks for cost optimization, performance, security, and reliability.

Settings

Manage connectors, feature flags, notification channels, and secret references.

Navigation

  • Overview - Aggregated view of runs, issues, advice, and tasks
  • Pipelines - Pipeline management and execution monitoring
  • Data Quality - Rule definition and health monitoring
  • Advisor - Recommendations and action tracking
  • Settings - System configuration and integrations

Models

  • control.pipeline - Pipeline definitions with stages/steps
  • control.run - Execution instances with event logs
  • control.dq.rule - Data quality rule definitions
  • control.dq.issue - Issue tracking with SLA support
  • control.advice - AI-generated recommendations
  • control.connector - External system integrations

Design System

Uses Microsoft Fluent 2 design tokens for consistent styling across Odoo and external applications.
