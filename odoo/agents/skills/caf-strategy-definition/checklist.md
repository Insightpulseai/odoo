# Checklist — caf-strategy-definition

- [ ] Adoption motivation identified (migrate / innovate / govern)
- [ ] Business outcomes defined with measurable metrics
- [ ] Fiscal outcomes quantified (cost reduction, revenue growth, cost avoidance)
- [ ] Agility outcomes defined (time to market, provisioning speed)
- [ ] Reach outcomes defined (geographic, access, scale)
- [ ] Engagement outcomes defined (user satisfaction, adoption rate)
- [ ] Cloud economics model created (TCO current vs target)
- [ ] OpEx vs CapEx shift documented
- [ ] Digital estate inventory completed
- [ ] Each workload classified using 5 Rs rationalization
- [ ] First adoption project identified with clear success criteria
- [ ] Organizational constraints and compliance requirements documented
- [ ] Skills gaps identified
- [ ] Risk register created with mitigations
- [ ] Strategy document produced in structured format
- [ ] Evidence captured in `docs/evidence/{stamp}/caf/strategy/`
