# palantir-foundry-integration-assessment

Assess whether Palantir Foundry integration is actually needed and which SDK to use — decision gate before any Palantir work begins.

## When to use
- Palantir integration proposal or request
- External platform evaluation involving Foundry
- Naming confusion detected between Microsoft Foundry and Palantir Foundry

## Key rule
Default status is **out-of-scope** unless explicitly needed. Never begin Palantir implementation before this assessment gate passes. Never confuse Microsoft Foundry (Azure AI Foundry) with Palantir Foundry — they are completely unrelated platforms from different companies.
