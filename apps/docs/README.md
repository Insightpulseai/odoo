# Odoo Documentation

Comprehensive documentation for Odoo 18/19 with IPAI Platform extensions, styled with Primer (GitHub's design system).

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3002
```

## Structure

- `/app` - Next.js app router pages
- `/components` - Reusable React components
- `/content` - MDX documentation content
- `/public` - Static assets

## Documentation Sections

- **Getting Started** - Installation and quick start guides
- **Developer** - ORM, Views, OWL, Security, Testing
- **Administrator** - Database, users, modules, performance
- **User** - Per-module user guides
- **Platform Kit** - Supabase, Edge Functions, RLS

## Tech Stack

- Next.js 16 (App Router)
- Primer React (GitHub's design system)
- MDX (Markdown + React components)
- TypeScript
