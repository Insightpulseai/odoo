# Brand Assets and Constants

Canonical brand tokens, color palettes, typography, and logo assets for the InsightPulse AI web properties.

## Convention

- Design tokens exported as CSS custom properties and TypeScript constants
- Logo assets in SVG format, PNG for social/OG only
- Color palette must match Figma design system source
- All web properties import from this package
