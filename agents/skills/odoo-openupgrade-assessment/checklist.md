# Checklist — odoo-openupgrade-assessment

- [ ] Current Odoo version and target version identified
- [ ] Full installed module list extracted from database or manifest
- [ ] OpenUpgrade branch for target version located and inspected
- [ ] Each module checked for migration script in OpenUpgrade
- [ ] Modules classified: covered / partial / missing
- [ ] Custom ipai_* modules inventoried separately
- [ ] Custom modules assessed for manual migration needs
- [ ] OCA modules checked for 19.0 branch availability
- [ ] Breaking changes in target version documented
- [ ] Risk classification assigned (low/medium/high/critical)
- [ ] Upgrade readiness report generated
- [ ] Evidence captured in `docs/evidence/{stamp}/oca-governance/odoo-openupgrade-assessment/`
