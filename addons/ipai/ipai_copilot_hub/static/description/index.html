IPAI Copilot Hub

Ops Control Room / AI Runbook Hub - A centralized operations dashboard that embeds an external Fluent UI application for managing AI runbooks, pipelines, and operational workflows.

Features

External Fluent UI Shell

Embeds a Microsoft Fluent UI React application in a seamless iframe integration.

Configurable URL

Hub URL is managed via system parameter ipai.copilot.hub_url.

Full-Screen Experience

Client action provides an immersive, full-screen control room interface.

Menu Structure

  • Ops Control Room (root menu - App tile)
    • Dashboard - Main embedded view
    • AI Runbooks - Runbook management
    • Configuration
      • Settings - Hub URL and display options

Configuration

Set the system parameter to configure the external hub URL:

  • ipai.copilot.hub_url - External Fluent UI app URL
  • ipai.copilot.hub_embed_mode - Embed mode (iframe/popup/redirect)
  • ipai.copilot.hub_show_toolbar - Show Odoo toolbar above hub

Architecture

This module acts as a bridge between Odoo and an external Ops Control Room application built with Microsoft Fluent UI React components. The external app is deployed separately (e.g., on Vercel) and loaded within Odoo's backend interface.
