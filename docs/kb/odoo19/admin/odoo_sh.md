# Odoo.sh Platform

## What it is

Odoo's PaaS (Platform as a Service) for hosting and developing Odoo projects.

## Key concepts

- **Branches:** Production, Staging, Development.
- **Builds:** Each commit triggers a build (instance).
- **Web Shell:** Access to the server via terminal.
- **Logs:** Real-time logging.

## Patterns

- **CI/CD:** Automated testing on every push.
- **Submodules:** Supported for private dependencies.
- **requirements.txt:** For Python dependencies.
