# Finance Seed Data — Canonical Source

## Single Source of Truth

The canonical source for all finance seed data is:

```
config/finance/Month-end Closing Task and Tax Filing (7).xlsx
```

All CSV files in this directory are **derived outputs** generated by:

```bash
python3 scripts/finance/export_seed_workbook.py \
  --workbook "config/finance/Month-end Closing Task and Tax Filing (7).xlsx"
```

**Rule**: Edit the workbook first, then regenerate CSVs. Never edit CSVs directly.

## Derived Outputs

| File | Rows | Source Sheet | Purpose |

## Date Normalisation

BIR filing dates in the workbook use a mix of 2025 and 2026 years. The export script
normalises all dates to the **2026 context year** (the fiscal year these filings target).
