# Month-End Close SOP

## Overview
This document outlines the standard operating procedure for month-end close.

## Deadlines
- **Prop** date: 3 days before month end.
- **Review** date: 2 days before month end.
- **Approval** date: Last day of month.

## Checklist
1. Review pending vendor bills.
2. Reconcile bank statements.
3. Post accruals.
4. Run deprecation.
