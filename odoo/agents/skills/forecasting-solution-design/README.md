# forecasting-solution-design

Designs demand forecasting architecture using Azure Databricks, covering scenario blueprints, feature engineering for time-series, evaluation methodology, and action integration.

## When to use
- Demand forecasting use case design (retail, FMCG, supply chain)
- Time-series prediction architecture review
- Inventory optimization or staffing prediction projects
- Analytics-to-action pipeline design

## Key rule
This is a scenario archetype, not a control-plane skill. It provides reusable patterns for forecast workloads but does not influence core platform architecture. Statistical methods should be the default unless ML provides measurable improvement.
