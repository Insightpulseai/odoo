# OCA Addons Directory

This directory holds OCA (Odoo Community Association) modules.

## Quick Clone

```bash
# Web addons (UI enhancements)
git clone --depth 1 -b 18.0 https://github.com/OCA/web.git web

# Server tools
git clone --depth 1 -b 18.0 https://github.com/OCA/server-tools.git server-tools

# Account financial tools
git clone --depth 1 -b 18.0 https://github.com/OCA/account-financial-tools.git account-financial-tools
```

## Recommended OCA Modules for InsightPulse AI

| Repo | Modules | Purpose |

| `server-tools` | base_automation | Advanced automation |
