# Examples — sap-analytics-ai-patterns

## Example 1: Unified analytics

**Input**: SAP guidance recommends unified analytics across ERP and external data

**Output**:
- Domain: Data integration
- Benchmark: Unified analytics platform connecting ERP data with external sources
- Source: Insights for SAP, Module 4
- Translation: Databricks Lakehouse with Odoo PG as source via Lakeflow Connect
- Platform mapping: SAP Analytics Cloud to Databricks + Superset
- Risk: Without unified analytics, ERP data remains siloed in Odoo PG only
