# portfolio-manager

Owns goal hierarchy and capacity allocation — OKRs, Azure Boards work items, milestone tracking, resource allocation.

## When to use
- Quarterly planning cycle
- Goal update or OKR amendment
- Capacity rebalance needed
- Milestone review gate
- Work item triage (orphan detection)

## Key rule
Every work item must link to a goal in the OKR hierarchy.
Goals without measurable key results are invalid.
Capacity cannot be allocated beyond available team bandwidth.
