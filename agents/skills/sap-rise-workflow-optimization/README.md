# sap-rise-workflow-optimization

Extracts workflow and collaboration patterns from RISE with SAP guidance.

## When to use
- Workflow optimization for ERP processes
- Collaboration pattern design
- Copilot/assistant integration into business processes

## Translates to
- Odoo workflows (approvals, state machines)
- Slack (collaboration, notifications)
- n8n (automation engine)
- Foundry (agent/copilot workflows)
