# Skill: Skill Trigger Tuning

## Purpose

Optimizes when a skill triggers — ensuring it fires on relevant prompts (recall) and doesn't fire on irrelevant ones (precision). Trigger quality is a first-class concern, not an afterthought.

## Owner Persona

`skill-author`

## Skill Type

`capability_uplift` — improves the precision/recall of skill triggering beyond naive matching.

## Key Principle

"A skill must trigger when it should and not overfire or underfire."
