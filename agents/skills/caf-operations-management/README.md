# caf-operations-management

Manages cloud operations with proper baselines and processes using CAF Manage phase methodology, covering management baseline, enhanced management, and business alignment through operational fitness reviews.

## When to use
- Operations baseline establishment for new environment
- Operational fitness review scheduled
- SLA/SLO definition or revision
- Monitoring or alerting gap identified
- Incident postmortem requiring operational improvement

## Key rule
Operations baselines must be achievable with current team capacity (solo developer with AI agent augmentation). SLA/SLO targets must be measurable and aligned with actual architecture capabilities. Monitoring must integrate with existing Azure Monitor and Application Insights.

## CAF Phase
Manage
