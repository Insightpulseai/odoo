# sap-btp-azure-integration

Extracts identity and integration patterns from SAP BTP on Azure guidance.

## When to use
- Identity provider integration design
- SSO architecture for multi-service platform
- Integration pattern selection

## Translates to
- Entra ID (identity provider)
- Odoo OIDC (ERP auth)
- Foundry (agent auth)
- MCP (automation auth)
