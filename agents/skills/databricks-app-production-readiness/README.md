# databricks-app-production-readiness

Assesses whether a Databricks App meets production-readiness criteria for serverless hosting, deployment discipline, authentication, and secure data access.

## When to use
- New Databricks App deployment
- App promotion to production
- App security or deployment review

## Key rule
Databricks "production ready" is a benchmark for production discipline, not a blanket trust label.
An app is production-grade only when: release maturity is GA or stable Public Preview, deployment is codified, health checks exist, and rollback is defined.
