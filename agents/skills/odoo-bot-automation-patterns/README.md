# odoo-bot-automation-patterns

Understand and validate OCA bot automation patterns — branch automation, migration issue creation, CI trigger patterns.

## When to use
- New series support is being planned
- Bot behavior needs review
- Automation pattern audit for alignment with OCA conventions

## Key rule
Never modify oca-github-bot configuration directly. Use OCA bot patterns as a benchmark
to understand and align custom CI/automation. The goal is pattern awareness, not direct
control of OCA infrastructure.
