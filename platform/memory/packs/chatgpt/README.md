# ChatGPT Pack

**Format**: Short, bullet-heavy, command-forward
**Token budget**: ~6,000 tokens max
**Optimized for**: Quick task execution, code generation, debugging

## Pack Contents

- `00_constitution.md` (from common) - Core operating principles
- `10_repo_map.md` (from common) - Repository structure
- `20_workflows.md` (from common) - Common commands
- `30_current_focus.md` - Active priorities and blockers
- `90_recent_changes.md` - Distilled recent activity

## Usage Pattern

1. Load pack at conversation start
2. Reference specific sections as needed
3. Keep responses actionable and concise
4. Produce executable commands, not guides

## Key Differences from Claude Pack

- Shorter context window assumed
- More terse explanations
- Focus on immediate execution
- Fewer nested tool calls
