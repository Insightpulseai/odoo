# Services: Helpdesk, Planning, Appointments

## What it is

Advanced service applications for support and scheduling.

## Key concepts

- **Helpdesk:** Ticket management, SLA, Canned Responses.
- **Planning:** Shift scheduling (Gantt view).
- **Appointments:** Online booking system.

## Patterns

- **Email Alias:** Create tickets from email.
- **Website Integration:** Appointment booking form on website.
