IPAI Agent Hybrid Orchestration
===============================

This module enables hybrid orchestration of InsightPulse AI agents across
cloud and on-premise environments.

Main features
-------------

* Route agent workloads between local and cloud execution targets.
* Preserve observability hooks for monitoring and logging.
* Designed to extend the core ``ipai_agent`` behaviors for hybrid use cases.

Configuration
-------------

* Install ``ipai_agent`` and configure hybrid endpoints as needed.
* Adjust routing preferences based on workload or compliance needs.
