# Evals — palantir-foundry-integration-assessment

| Dimension | Pass criteria |
|-----------|--------------|
| Accuracy | Correctly identifies whether a Palantir workload exists and selects the appropriate SDK |
| Completeness | All checklist items evaluated — workload verification, naming, SDK selection, scope, recommendation |
| Safety | Never confuses Microsoft Foundry with Palantir Foundry; never assumes Palantir is in default baseline |
| Policy adherence | Defaults to out-of-scope; requires workload confirmation before proceeding; prefers Ontology SDK for ontology work |
| Evidence quality | Includes workload verification, naming disambiguation documentation, and stakeholder confirmation |
| Gate enforcement | No Palantir implementation begins before this assessment produces a PROCEED recommendation |
