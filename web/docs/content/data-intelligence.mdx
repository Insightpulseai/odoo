# Data Intelligence

The analytics plane runs on Databricks with Unity Catalog for governed data engineering.

## Lakehouse Architecture

Medallion pattern: **Bronze → Silver → Gold → Platinum**

| Layer | Purpose | Storage |
|-------|---------|---------|
| Bronze | Raw ingestion (JDBC extract from Odoo PG) | Delta Lake |
| Silver | Cleaned, validated, deduplicated | Delta Lake |
| Gold | Business-ready aggregates and dimensions | Delta Lake |
| Platinum | ML features, scoring tables, API-ready | Delta Lake |

## Data Sources

| Source | Extract Method | Frequency |
|--------|---------------|-----------|
| Odoo PostgreSQL | JDBC (Databricks connector) | Hourly / Daily |
| Azure Blob Storage | Auto Loader | Event-driven |
| n8n webhook payloads | REST → Bronze | Real-time |

## Key Pipelines

- **Finance Pipeline**: GL entries → Silver → Gold (P&L, BS, Cash Flow)
- **HR Pipeline**: Payroll, timesheets → Silver → Gold (utilization, cost)
- **Project Pipeline**: Tasks, milestones → Gold (burndown, profitability)

## Reporting

- **Primary**: Power BI (mandatory business-facing surface)
- **Supplemental**: Apache Superset (self-hosted, internal analytics)
- **Governance**: Unity Catalog for lineage, access control, data quality
