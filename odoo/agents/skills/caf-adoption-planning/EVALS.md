# Evals — caf-adoption-planning

| Dimension | Pass criteria |
|-----------|--------------|
| Accuracy | Digital estate inventory matches actual deployed resources |
| Completeness | All workloads classified, all waves sequenced, skills gaps identified |
| Feasibility | Plan is achievable given team size (solo developer) and budget constraints |
| Organizational fit | Organizational model reflects actual team structure, not large-enterprise assumptions |
| Evidence quality | Inventory based on real resource data, not hypothetical estimates |
| Dependency awareness | Wave sequencing respects workload dependencies — no orphaned migrations |
