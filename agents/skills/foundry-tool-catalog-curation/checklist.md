# Checklist — foundry-tool-catalog-curation

- [ ] Tool type classified (local MCP, remote MCP, native Foundry)
- [ ] Auth mode assessed and compared to minimum requirements
- [ ] Provider trust level evaluated
- [ ] Category fit checked against approved categories
- [ ] Checked against `ssot/agents/mcp-baseline.yaml` for conflicts/duplicates
- [ ] Verdict issued: approved, conditional, or forbidden
- [ ] Rationale documented for the verdict
- [ ] Conditions documented (if conditional verdict)
- [ ] Local vs remote MCP distinction explicitly stated
- [ ] Auth mode requirement documented for remote MCP tools
- [ ] Trust boundary assessment documented
- [ ] Evidence captured in `docs/evidence/{stamp}/foundry/tool-catalog/`
