- [ ] Branch name follows convention (feat/, fix/, chore/, docs/, test/)
- [ ] Commit messages follow OCA format (type(scope): description)
- [ ] All required status checks are passing (lint, typecheck, test, build)
- [ ] CODEOWNERS file covers all modified paths
- [ ] No force-push detected on protected branches

- [ ] GHAS secret scanning shows no alerts on diff
- [ ] No Enterprise module dependencies introduced
- [ ] PR description references relevant spec bundle or issue
- [ ] Evidence captured in `docs/evidence/{stamp}/odoo-delivery/odoo-github-flow/`
