IP Expense MVP
==============

This module delivers a minimal expense tracking and approval workflow
for InsightPulse on Odoo CE.

Main features
-------------

* Capture employee expenses with basic categorization.
* Route submissions for manager review and approval.
* Provide a starting point for broader finance automation.

Configuration
-------------

* Install the module and grant expense access to relevant users.
* Configure any approval rules or categories as needed.

Roadmap
-------

This README is intentionally concise to satisfy OCA CI checks and will
be expanded as the MVP evolves.
