# .github

Org-wide GitHub governance, reusable automation, repository standards, and bootstrap templates for InsightPulseAI.

## Purpose

This repository is the GitHub control surface for the organization. It owns shared workflows, reusable actions, issue and PR templates, repository defaults, and policy enforcement that should apply consistently across all repos.

## Owns

- Reusable GitHub Actions workflows
- Shared composite actions
- Issue and PR templates
- Org profile and contribution entrypoints
- GitHub-side policy checks and validation
- Repository bootstrap templates

## Does Not Own

- Product code
- Runtime infrastructure
- Per-repo application configuration
- Azure deployment logic specific to one workload

## Authority Model

- GitHub is the source of truth for: repositories, pull requests, CI workflows, repository policy automation
- Azure DevOps is the source of truth for: portfolio planning, selected deployment orchestration where applicable

## Validation

Changes in this repo must:

- validate workflow syntax
- validate action references
- avoid policy drift across downstream repos
- preserve compatibility with the org authority model
