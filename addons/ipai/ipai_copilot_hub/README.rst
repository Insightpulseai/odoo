A centralized operations control room that embeds an external Fluent UI
application for managing AI runbooks, pipelines, and operational workflows.

Features
========

* Embeds external Fluent UI shell via iframe
* Configurable external URL via system parameter
* Full-screen client action for immersive experience
* OWL component with loading states and error handling
* Fluent 2 design tokens integration

Architecture
============

This module acts as a bridge between Odoo and an external Ops Control Room
application built with Microsoft Fluent UI React components.

The module provides:

1. **Client Action**: A full-screen OWL component that renders the external
   Fluent UI application in an iframe.

2. **Configuration**: System parameter ``ipai.copilot.hub_url`` to configure
   the external application URL.

3. **API Endpoint**: A controller that provides the hub URL and any required
   authentication tokens to the frontend.

Configuration
=============

1. Navigate to Settings > Technical > System Parameters
2. Find or create the parameter ``ipai.copilot.hub_url``
3. Set the value to your Fluent UI application URL
   (default: ``https://ops.insightpulseai.net``)

Usage
=====

1. Install the module via the Apps menu or CLI
2. Configure the hub URL (see Configuration)
3. Navigate to **Ops Control Room** in the main menu
4. The external Fluent UI application will load in the main content area

Menu Structure
--------------

* **Ops Control Room** (root menu)
    * AI Runbooks - Main embedded dashboard
    * Settings - Configuration and preferences
