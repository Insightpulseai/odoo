# caf-adoption-planning

Creates cloud adoption plan with timeline, digital estate inventory, skills readiness assessment, and organizational alignment using CAF Plan phase methodology.

## When to use
- Cloud strategy is approved and ready for execution planning
- New adoption wave is being initiated
- Organizational restructuring affects cloud team composition
- Skills gaps are identified in cloud operations

## Key rule
Every adoption plan must include a digital estate inventory, sequenced migration waves, skills readiness assessment, and organizational alignment model. Plans must be realistic given team size and budget constraints.
