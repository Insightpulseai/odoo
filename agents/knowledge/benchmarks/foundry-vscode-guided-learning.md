# Foundry VS Code Guided Learning Benchmark

## Source

https://learn.microsoft.com/en-us/azure/foundry/how-to/develop/get-started-projects-vs-code

## Weight

0.95

## Why it matters

Official Microsoft surface for learning Foundry projects inside VS Code:
- Open projects and navigate Resources / Tools / Help
- Deploy models from Model Catalog
- Generate starter code for deployed models
- Use Model Playground for prompt testing
- Use Agent Playgrounds (remote and local) and Local Visualizer
- Deploy hosted agents from the extension

## Allowed influence

- Learning path design for agent builders
- Project-onboarding flow
- Model deployment onboarding sequence
- VS Code training sequence for agent-platform contributors

## Must not influence

- Repo topology
- Odoo module boundaries
- Platform SSOT structure
- Production runtime architecture decisions
