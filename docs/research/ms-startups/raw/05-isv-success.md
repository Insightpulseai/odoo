# ISV Success — full benefits structure

## Core Package (free year 1)
Azure:
- $5,000 Azure sponsorship (usage credits; NOT transferable to MACC subscriptions)
- Azure Standard Support plan

Product:
- M365 E5 developer subscription
- Dynamics 365 Sales/Field/Customer Service partner sandbox
- Dynamics 365 discounted + free licenses
- Visual Studio Enterprise subscription
- GitHub Enterprise Cloud (YEAR 1 ONLY — not renewed year 2+)
- Limited-access features for Azure AI Services

1:1 support:
- Go-to-market sessions
- Application Design Session (ADS)
- Application Design Review (ADR)
- App publishing consultation
- AI technical consultation
- Marketplace listing optimization

## Expanded Package (invitation only, based on MBS + ACR)
- +$20,000 Azure ($25k total with Core)
- 50 hours of 1:1 technical consultations
- Eligible for +$5,000 Azure if you use GitHub Copilot during acknowledgement

## Advanced Package (Nov 18, 2024, Certified Software Designation required)
- +$25,000 Azure ($50k total with Expanded + Core)
- Up to $100K cash for Azure AI/Analytics apps
- Up to $50K cash for customer migration (Marketplace Rewards portion)
- All Expanded Package benefits included

## Renewal
- $1,550 USD/year starting year 2
- Must publish new offer OR upgrade existing to transactable
- GitHub Enterprise NOT included in renewal year
- Renewal window opens 30 days before end date

## Cross-program stacking
- CAN stack Founders Hub + ISV Success simultaneously
- CANNOT apply ISV Success Azure sponsorship to a subscription with another active sponsorship (e.g., Microsoft for Startups sponsorship). Must use alternate EA/PAYG sub or wait for previous sponsorship to expire.

## Transactable professional services (Sep 1, 2024+)
- Available in USA, UK, Canada ONLY
- Cannot enroll in ISV Success on prof-services alone — must have a B2B app
- MBS from prof-services counts toward Marketplace Rewards if attached to a transactable app
