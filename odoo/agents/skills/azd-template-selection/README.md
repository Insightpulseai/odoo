# azd-template-selection

Select the appropriate Azure Developer CLI template for a given workload type.

## Owner

azure-bootstrap-engineer

## When to use

- New application workload needs provisioning
- Choosing between azd templates for a technology stack
- Evaluating template compatibility with the platform

## Key principle

Templates are implementation fixtures, not architecture doctrine. Select for secure-by-default posture, adapt to the canonical platform stack.
